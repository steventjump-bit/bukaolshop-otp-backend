# Backend OTP Bukaolshop

Backend sederhana untuk menghasilkan dan memverifikasi kode OTP (One-Time Password) untuk aplikasi Bukaolshop.

## Fitur Utama

*   Menghasilkan kode OTP unik
*   Memverifikasi kode OTP yang dimasukkan pengguna
*   Konfigurasi mudah
*   Keamanan terjamin

## Teknologi yang Digunakan

*   Python
*   Flask
*   PostgreSQL
*   Cryptography
